# analise-biomecanica-stiff
aplicação para simular e analisar o exercício Stiff (Stiff-Legged Deadlift), o foco deve estar na qualidade do movimento (segurança da coluna e ativação dos isquiotibiais) e na progressão de carga (gestão de volume/intensidade).

# O programa é dividido em 3 funções
1- Dados do Paciente
O usuario podera manipular os dados do peso (kg), altura do paciente (m) e a carga da barra alocada (kg)

2- Upload
O video do exercicio feito pelo paciente é enviado e analisado, calculando os graus da flexão e o torque realizado

3- Relatorio Biomecânico
Na terceira e ultima função, após a analise, o programa retorna um relatorio em formato de grafico de ambas a força Torque aplicada na Lombar(N.m) e Força de Cisalhamento(N)
tendo como opção exportar os dados para uma planilha do excel para comparação e analise de outros pacientes.
